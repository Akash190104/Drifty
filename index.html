DRIFTY

Drifty is an open-source interactive File Downloader system built with java. It takes the link to the file, the directory where it needs to be saved and the filename of the downloaded file as input and downloads it.

more info

More Information About Drifty

It is currently available in CLI mode. It uses URL class for connecting to the server to get the file, FileSystems, Files and Path classes to check existence of the directory and create them if needed, and ReadableByteChannel and FileOutputStream clases for downloading the file.
